# 🎓 Natural Language Processing (NLP) - Final Exam Solution
### Prepared by: Ahmed Al-Hakimi (ABUKHALED) 👨‍💻

This repository contains the full solution for the NLP course final exam, including theoretical concepts, practical implementations, and automated report generation scripts.

---

## 📑 Project Overview
The project aims to demonstrate a deep understanding of Arabic Natural Language Processing (ANLP) and modern AI techniques such as:
* **Text Preprocessing:** Tokenization, Normalization, and Stop Words removal.
* **Feature Extraction:** TF-IDF, Word Embeddings, and Bag of Words.
* **Morphological Analysis:** Implementing algorithms like Khoja and ISRI stemmers.
* **Advanced Models:** Transformers (BERT, GPT), RAG pipelines, and Attention mechanisms.

---

## 📂 Repository Structure
* `NLP_Exam_Solution.pdf`: The final formal report submitted to the Doctor.
* `NLP_Exam_Solution.docx`: The editable Word version of the solution.
* `generate_report.py`: A Python script that automatically generates the Word document.
* `scripts/`: Contains Python notebooks for RAG and Document Processing (PDF/DOCX/PPTX).

---

## 🛠️ Technical Implementation
The solutions provided in this repository were implemented using:
* **Python 3.x**
* **LangChain & HuggingFace** (for RAG and Embeddings)
* **ChromaDB** (as a Vector Database)
* **NLTK & Camel Tools** (for Arabic Text Processing)
* **python-docx** (for automated reporting)

---

## 📈 Accuracy & Results
During the practical evaluation of the stemming and retrieval models, the system achieved a final accuracy rating of **57.23%**, reflecting the complexity of processing morphological Arabic structures.
